# cpp-prebuilt-libs

## 简介

Prebuilt C/C++ Libraries.
Automated builds with GitHub Actions, providing binary releases and build scripts for popular C/C++ libraries.

预编译的 C/C++ 第三方库。
通过 GitHub Actions 自动构建，为流行的 C/C++ 库提供二进制发布和编译脚本。

## 库清单
操作系统与编译器组合：
- 1. Linux + GCC
- 2. Linux + Clang
- 3. Windows + MSVC
- 4. Windows + MinGW-w64
- 5. Windows + MSYS2 MinGW
- 6. Windows + Cygwin
- 7. macOS + XCode
